# Crimson AI Nexus (Universal Headless Agent Core)

## 🩸 Core Orchestrator: The Crimson Agent
- **Universal Operational Scope:** The Master Overseer across the repository. It intercepts high-level developer prompts from Gemini, determining whether to build a new financial tool, a video-generation app, or a social media pipeline.
- **Intelligent Context Router:** Dynamically analyzes commands, scans the local asset tracking logs (`database/assets.json`), and patches existing tools rather than generating duplicate systems.
- **Multimodal Processor:** Ingests visual strategy files (chart screenshots, workflow diagrams, design mockups) and translates them into deployable code structures and logic.

## ⚙️ Worker Submodules (The Distributed Village)
- **The Software Builder (`agents/agent-zero` & `agents/aider`):** Executes low-level app generation, API routing, dependency compilation, and code patching.
- **The Media Engine (`agents/video-factory`):** Automates scripting, AI voice generation, thematic overlays, and horizontal/vertical MP4 serialization pipelines.
- **The Distribution Node (`agents/social-poster`):** Controls external API channels to authenticate, schedule, and distribute generated media to external platforms automatically.

## 📉 Self-Correction & Enforcement Guardrails
- **Reinforcement Learning:** Embeds mathematical optimization models (PPO) in all trading/algorithmic instances to self-correct and adjust coefficients after paper-trading simulations.
- **Strict Logic Enforcements:** When building financial dashboards, the system inherently prioritizes API authentication sequences (e.g., establishing 'Read' and 'Trade' token authorizations) prior to rendering user interfaces or balance displays.
